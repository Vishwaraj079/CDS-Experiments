# CDS-Experiments
Data Structure &amp; C++ Lab Experiments
| No. | Experiment Title                            | Folder Name                      | Description                                                                  |
| --- | ------------------------------------------- | -------------------------------- | ---------------------------------------------------------------------------- |
| 1   | **Basic I/O Operations / Basic Calculator** | `Exp1_BasicIO_Calculator/`       | Demonstrates basic input/output and arithmetic operations in C++.            |
| 2   | **Data Types**                              | `Exp2_DataTypes/`                | Displays various C++ data types, sizes, and example values.                  |
| 3   | **Operators**                               | `Exp3_Operators/`                | Programs on arithmetic, logical, and conditional operators.                  |
| 4   | **Bitwise Operators**                       | `Exp4_BitwiseOperators/`         | Demonstrates set/reset bit manipulation using bitwise operators.             |
| 5   | **Conditional Statements**                  | `Exp5_ConditionalStatements/`    | Uses if/else to check even/odd, vowel/consonant, largest number.             |
| 6   | **Loops (For & While)**                     | `Exp6_Loops/`                    | Programs using loops for repetition and pattern printing.                    |
| 7   | **Arrays (1D)**                             | `Exp7_Array/`                    | Programs for array input/output, reversing, search, sum, min/max.            |
| 8   | **2D Arrays (Matrix Operations)**           | `Exp8_2DArray/`                  | Programs on matrix input, addition, multiplication, transpose, etc.          |
| 9   | **Pointers**                                | `Exp9_Pointers/`                 | Demonstrates pointer arithmetic and pointer-based operations.                |
| 10  | **Pointer Operations**                      | `Exp10_PointerOperations/`       | Programs using call by value, reference, and pointer manipulation.           |
| 11  | **Classes and Objects**                     | `Exp11_ClassesObjects/`          | Basic OOP concepts with multiple examples using classes and objects.         |
| 12  | **Constructors & Destructors**              | `Exp12_ConstructorsDestructors/` | Demonstrates default, parameterized, copy constructors and destructors.      |
| 13  | **Overloading**                             | `Exp13_Overloading/`             | Function, operator, and constructor overloading in C++.                      |
| 14  | **Inheritance**                             | `Exp14_Inheritance/`             | Programs showing single, multiple, multilevel, and hierarchical inheritance. |
| 15  | **Recursion**                               | `Exp15_Recursion/`               | Recursive functions for factorial, sum till n, and string reversal.          |
| 16  | **Exception Handling**                      | `Exp16_ExceptionHandling/`       | Try-catch blocks for divide-by-zero and invalid-age exceptions.              |
| 17  | **Linked List**                             | `Exp17_LinkedList/`              | Implementation of singly linked list operations.                             |
| 18  | **Queue Implementation**                    | `Exp18_Queue/`                   | Queue operations (enqueue, dequeue) using arrays.                            |
| 19  | **Stack Implementation**                    | `Exp19_Stack/`                   | Stack operations (push, pop, display) using arrays.                          |
| 20  | **Searching Algorithms**                    | `Exp20_Searching/`               | Linear, sequential, and binary search implementations.                       |
| 21  | **Sorting Algorithms**                      | `Exp21_Sorting/`                 | Selection, bubble, and quick sort algorithms.                                |
| 22  | **Algorithm Analysis (Big O)**              | `Exp22_BigO/`                    | Study of time complexity using example algorithms.                           |
